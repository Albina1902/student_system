# 🎓 Student Management System (Django)

Веб-приложение для управления студентами, группами и курсами, разработанное на **Django**.  
Проект реализует полный CRUD, авторизацию пользователей и базовую учебную логику.

---

## 🚀 Функциональность

### 👤 Управление студентами
- Добавление, редактирование, удаление
- Просмотр списка студентов
- Привязка студента к группе
- Хранение email и даты рождения

### 👥 Управление группами
- Создание и редактирование групп
- Привязка к факультету
- Просмотр студентов в группе

### 📚 Управление курсами
- Создание курсов
- Назначение преподавателя (User)
- Просмотр списка курсов

### 📝 Оценки (опционально)
- Привязка оценки к студенту и курсу
- Хранение даты выставления оценки

### 🔐 Авторизация
- Используется стандартная система Django Auth
- Роли:
  - **Администратор** — полный доступ
  - **Преподаватель** — работа с курсами и оценками
